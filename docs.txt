SQL запросы - CREATE, INSERT, SELECT, UPDATE, DROP

Оператор SQL CREATE:
                  Служит для создания обектов базы данных. ((Разные СУБД работают с разными обьектами,
но наиболее с обшими СУБД будут команды создания ( ОПЕРАТОР - SQL CREAT TABLE)))



Оператор SQL INSERT:
                  Оператор языка SQL, который позволяет добавить строки в таблицу, заполняя их значениями.
Значения можно вставлять перечислением с помощью слова (values) и перечислив их в круглых скобках через запятую.



Оператор SQL SELECT:
                    Ключевое слово, которое сообщает базе данных о том, что оператор является запросом.
Все запросы начинаются с этого слова, за ним следует пробел (Column_list (*), FROM table_name "[]")



Оператор SQL UPDATE - изменяет имеющиеся данные в таблице.



Оператор SQL DROP - является оператором удаление базы данных.
